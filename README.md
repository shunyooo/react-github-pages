## Feature

## Use as Template

####  `init.sh`

- Rename app dir to top dir name
- Change app settings for the current github repository
- Build and deploy



## DEV

#### `npm start`

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.



## DEPLOY

#### `npm run deploy`

Push & deploy to the `gh-pages` branch
